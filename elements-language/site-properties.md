# Site Properties

Site properties can be accessed from all templates files.



**Title**&#x20;

Plain text value.

```
%id=rw:projectTitle%
```

**Slogan**&#x20;

Plain text value.

```
%id=rw:projectSlogan%
```

**Footer**

Plain text value.

```
%id=rw:projectFooter%
```

**Email**

Plain text value.

```
%id=rw:projectEmail%
```

**Contact Text**

Plain text value.

```
%id=rw:projectContactText%
```

**Base URL**

Plain text value.

```
%id=rw:projectBaseURL%
```

**Enable Social Tags**

Bool, returns a string of "True" or "False".

```
%id=rw:projectEnableSocialTags%
```

**Twitter Account**

Plain text value.

```
%id=rw:projectTwitterAccount%
```

**Project Language**

Plain text value.

```
%id=rw:projectLanguage%
```

**Banner URL**

Plan text value.

```
%id=rw:projectBannerURL%<
```

**Banner Alt Text**

Plan text value.

```
%id=rw:projectBannerAltText%
```

**Logo URL**

Plan text value.

```
%id=rw:projectLogoURL%
```

**Logo Alt Text**

Plan text value.

```
%id=rw:projectLogoAltText%
```

### Code Example

The following example outputs the project title and slogan.

```
<H1>%id=rw:projectTitle%</h1>
<h2>%id=rw:projectSlogan%</h2>
```
