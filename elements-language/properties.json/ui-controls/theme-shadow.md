# Theme Shadow

Displays the Theme Studio Shadow control.

{% tabs %}
{% tab title="Control Example" %}
```json
{
    "title": "Shadow",
    "id": "boxShadow",
    "themeShadow": {
        "default": {
            "name": "none"
        }
    }
}
```
{% endtab %}

{% tab title="Group Example" %}
```json
{
    "groups": [{
        "title": "Theme Shadow Example",
        "properties": [{
            "title": "Shadow",
            "id": "boxShadow",
            "themeShadow": {
                "default": {
                    "name": "none"
                }
            }
        }]
    }]
}
```
{% endtab %}
{% endtabs %}
