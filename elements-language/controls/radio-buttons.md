# Radio Buttons

Radio Buttons can be used to choose a value from an array of values.



```
{
    "buttons": {
    "style": "radio",
    "items": [
        {
        "subtitle": "Left",
        "value": "left",
        "icon": "fa-align-left",
        "default": true
        },
        {
        "subtitle": "Center",
        "value": "center",
        "icon": "fa-align-center"
        },
        {
        "subtitle": "Right",
        "value": "right",
        "icon": "fa-align-right"
        },
        {
        "subtitle": "Justify",
        "value": "justify",
        "icon": "fa-align-justify"
        }
    ]
    },
    "format": "text-{{value}}",
    "title": "Align2",
    "property": "textAlign2"
},
```
