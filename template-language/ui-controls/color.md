# Color



### Standard Color Picker (color)

The Color control displays the standard macOS color well.

```
{
	  "title" : "Colors",
	  "property" : "colors",
	  "colors" : {
		  "items" : [{
			  "subtitle" : "Quote",
			  "default" : "#616161",
			  "property" : "colors-one"
		  }, {
			  "subtitle" : "Citation",
			  "default" : "#BFBFBF",
			  "property" : "colors-two"
		  }]
	  }
  },
```
