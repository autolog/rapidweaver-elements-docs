### Spacing Example

You can use the following code as a starting point for adding the padding conrol.

```
{
    "title": "Margin",
    "property": "margin",
    "format": "m{{side}}-[{{value}}]",
    "themeSpacing": {
    "default": {
              "top": "0",
              "right": "auto",
              "bottom": "0",
              "left": "auto"
            }
    }
}

```
