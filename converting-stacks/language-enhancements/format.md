# Format



### Code Examples

The following code use the `format` property to transform a value into a valid tailwind css class.

```
{
          "visible": {
            "id": "customSizing",
            "value": true
          },
          "title": "Top",
          "property": "paddingTop",
          "format": "pt-{{value}}",
          "default": "2",
          "slider": {
            "use": "Spacing"
          }
        },
```
